yii2-widget-sweet-alert
----------

Yii2 widget wrapper for sweet-alert

## Install

```
composer require light/yii2-widget-sweet-alert=dev-master
```

## Usage

```
use light\widgets\SweetAlert;

SweetAlert::widget([
	'title' => 'hello',
	'text' => 'this is discription for alert',
	....
]);

```
